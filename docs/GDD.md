## 一、遊戲概述

### 1.1 基本資訊

| 項目       | 內容 |
| :---       | :--- |
| **遊戲名稱** | 台灣三國誌 (Chronicles of Formosa: Online SLG) |
| **類型**     | 大型多人線上歷史策略遊戲 (MMOSLG) |
| **平台**     | Desktop (Windows / macOS / Linux) |
| **引擎**     | Go + Ebiten 2D |
| **同服人數** | 10,000 人 |
| **賽季週期** | 三個月一循環 |

### 1.2 核心設計哲學

1. **天災是機會而非純粹懲罰**：災後可能發現古隧道、新水源，農產加成。
2. **族群衝突是動態平衡而非死局**：緊張氣氛儀可視化，玩家有預防操作空間。
3. **社會關係決定存亡**：萬人環境下單打獨鬥必死，系統引導玩家「結庄」。
4. **歷史沉浸而非歷史枷鎖**：半史實半架空，三大勢力共存於同一舞台。

### 1.3 目標受眾

- 喜愛歷史策略遊戲的玩家（如：三國志系列、信長之野望）
- 對台灣歷史有興趣的玩家
- 喜愛大型社交策略（如 EVE Online）的硬核 SLG 玩家

## 二、世界觀與歷史背景

### 2.1 時空設定

遊戲建立在「半史實、半架空」的舞台。歷史上「大肚王國」、「朱一貴」與「清廷」並非始終處於同一時空的巔峰，但本作透過**平行時空**設定，將三股勢力彙整於同一張地圖上。

### 2.2 地理範圍

| 項目       | 範圍 |
| :---       | :--- |
| **本島**   | 北起雞籠（基隆），南至鵝鑾鼻 |
| **海域**   | 周邊 80 海浬，含澎湖群島 |
| **關鍵水道** | 黑水溝（台灣海峽） |
| **地形分區** | 西部平原、中央山脈、東部後山、北部丘陵 |

### 2.3 歷史背景深度

#### 大肚王國（1540s–1732）
- 台灣中部最大的原住民聯邦政體
- 由巴布拉族（拍瀑拉族）領導，統轄十六社
- 鼎盛時期曾擊退荷蘭東印度公司的北征

#### 朱一貴事件（1721）
- 清代台灣規模最大的民變之一
- 朱一貴自稱「中興王」，建國號「大明慈悲國」
- 閩粵族群矛盾導致義軍內部分裂，最終被清軍平定

#### 清朝台灣治理（1684–1895）
- 以府城（台南）為行政中心
- 常駐兵力不足，高度依賴福建援軍
- 「三年一小反，五年一大亂」的治理困境

## 三、三大陣營設計

### 3.1 陣營總覽

| 陣營             | 代表人物        | 核心特性      | 政治結構     |
| :---             | :---            | :---          | :---         |
| **大清台灣府**   | 藍廷珍、施世驃  | 帝國權威      | 官僚體系     |
| **中興義民盟**   | 朱一貴、杜君英  | 漢民怒火      | 秘密結社     |
| **大肚王國**     | 甘仔轄·阿拉米  | 祖靈守護      | 鬆散聯邦制   |

### 3.2 大清台灣府

#### 核心能力：帝國後援
- 可從福建徵調物資與重型火砲
- 掌握府城，擁有穩定的後勤補給線
- 解鎖「京城高級官位」與「精銳禁衛軍」

#### 特色兵種
| 兵種       | 特性 |
| :---       | :--- |
| 藤牌兵     | 高防禦，擋遠程攻擊效果顯著 |
| 綠營鳥銃手 | 遠程火器部隊，對無甲目標傷害極高 |
| 福船水師   | 海上霸主，封鎖海域能力最強 |

#### 內部弱點
- 官僚腐敗：隨機觸發「貪汙事件」降低士氣
- 援軍受限：黑水溝湍流值影響福建增援時效
- 地方控制力弱：遠離府城的區域容易失控

### 3.3 中興義民盟

#### 核心能力：羅漢招募
- 流民（羅漢腳）可瞬間轉化為兵力
- 擅長快速擴張與煽動城市民變
- 從無到有的「開國登基」成就感

#### 特色兵種
| 兵種         | 特性 |
| :---         | :--- |
| 鴨母軍       | 農兵轉化，成本極低但訓練度差 |
| 六堆客勇     | 客家民兵，防禦型精銳 |
| 天地會死士   | 敢死突擊隊，夜襲加成極高 |

#### 內部弱點
- 易內訌：閩粵族群矛盾可能觸發內部分裂
- 煽動力強但組織鬆散：沒有穩定的行政體系
- 後勤薄弱：缺乏穩定的物資生產鏈

### 3.4 大肚王國

#### 核心能力：祖靈守護
- 無視瘴氣，山林移動力不減
- 中部獵場掌控權，隱蔽伏擊能力最強
- 自然災害威力減免（生態靈力機制）

#### 特色兵種
| 兵種     | 特性 |
| :---     | :--- |
| 出草勇士 | 夜間攻擊力 +100%，恐懼效果 |
| 鏢槍手   | 中遠程投射，對輕甲高效 |
| 巫師     | 支援型，提供區域 BUFF / DEBUFF |

#### 內部弱點
- 鬆散聯邦：十六社意見不一，需要議事廳協調
- 科技劣勢：無法自產火器
- 人口限制：不像漢民可以大量招募流民

### 3.5 陣營動態平衡機制

#### 3.5.1 入陣激勵
系統實時監測各陣營人數：
- 人數最少的陣營獲得**「天命加成」**：
  - 開墾速度 +30%
  - 部隊攻擊力 +15%
  - 資源產出 +20%

#### 3.5.2 內鬨系統 (Civil War)
當單一陣營人數超過全服 40%，自動啟動內部競爭：
- **清軍**：官場派系鬥爭（主戰派 vs 主和派）
- **義軍**：漳州派 vs 泉州派 vs 客家派
- **大肚王國**：十六社正統之爭

內部分裂會消耗陣營總資源，直到有人統一天下。

#### 3.5.3 NPC 帝國浪潮 (The Great Tide)
- 若無玩家選清軍 → 清軍轉為強大 AI 侵略者
- 若島內勢力過強 → 觸發「大清南巡」或「紅毛番復仇」
- 強大 NPC 艦隊封鎖海域，迫使所有玩家合作防禦

## 四、核心玩法系統

### 4.1 庄頭共享經營 (Joint Development)

庄頭系統是萬人版的核心創新，解決 10,000 名玩家的空間擠壓問題。

#### 4.1.1 庄頭結構
- **地圖格位**：一個「庄頭」可容納最多 **100 名**玩家
- **大地圖呈現**：顯示 100 個繁榮庄頭，而非 10,000 個散亂小屋
- **庄頭等級**：根據投入人口與資源，庄頭可升級（草寮→竹屋→磚造→城寨）

#### 4.1.2 職位分工
| 職位     | 責任 | 人數上限 |
| :---     | :--- | :---     |
| 庄長     | 內政決策、對外外交、發起救災 | 1 |
| 墾首     | 帶隊向瘴氣區擴張土地 | 5 |
| 武師/勇士 | 訓練民兵與防禦出草 | 10 |
| 商賈     | 管理糧食與銀兩貿易 | 5 |
| 庄民     | 基礎勞動力、資源生產 | 79 |

#### 4.1.3 庄頭推舉制度
- 庄長由庄內玩家投票選出，任期為遊戲內 30 天
- 若民忠低於 30，可發起「彈劾投票」
- 庄長可指定副庄長，離線時代理決策

### 4.2 精力值系統 (Stamina System)

解決「24 小時在線碾壓」問題：

| 項目         | 說明 |
| :---         | :--- |
| **每日上限** | 100 精力值 |
| **恢復速率** | 每小時恢復 5 點 |
| **內政消耗** | 開墾 10 / 建築 15 / 研發 20 |
| **軍事消耗** | 行軍 5/格 / 攻城 30 / 偵查 5 |
| **救災消耗** | 支援庄長 10 / 個人救災 20 |

### 4.3 天災救災：【牛車糧道守衛】

#### 4.3.1 觸發條件
- 伺服器同步觸發地震或颱風
- 災害發生前 1 小時，`GlobalNavbar` 顯示「氣候警告」閃爍動畫

#### 4.3.2 小遊戲玩法
1. **場景**：俯瞰圖，多個受災村落（需求點）與玩家倉庫（起始點）
2. **操作**：限時內繪製牛車行進路線，將物資送達各村落
3. **障礙**：
   - 土石流（阻斷道路）
   - 亂民（搶糧，需護衛隊排除）
   - 斷橋（需修橋工具）
4. **參與方式**：
   - 庄長：操作小遊戲本體
   - 庄民：點擊「支援」，將精力值轉給庄長，增加可用道具

#### 4.3.3 獎勵機制（化災為利）
| 結果         | 獎勵 |
| :---         | :--- |
| **完美救災** | 「災民感念」BUFF，三天農業 +50% |
| **一般救災** | 資源損失減半 |
| **重建發現** | 機率發現「古隧道」或「新水源」，永久提升格位價值 |
| **救災失敗** | 民忠大幅下降，可能觸發逃荒 |

### 4.4 族群緊張儀 (Ethnic Tension System)

#### 4.4.1 緊張值機制
- 城市介面上方顯示一條燃燒的**「火藥引信」**
- 引信長度代表距離「分類械鬥」爆發的時間

#### 4.4.2 緊張值變動因素
| 因素 | 效果 |
| :--- | :--- |
| 族群比例失衡（閩>70% 或 粵>70%） | 緊張值 +5/天 |
| 資源短缺（糧食<30%） | 緊張值 +3/天 |
| 治安低下（官兵不足） | 緊張值 +2/天 |
| 辦桌（消耗糧食） | 緊張值 -10 |
| 共同祭祀（消耗銀兩） | 緊張值 -15 |
| 聯姻（兩族群玩家協議） | 緊張值 -20 |
| 出借牛隻（跨族群資源共享） | 緊張值 -8 |

#### 4.4.3 視覺回饋層級
| 緊張等級 | 引信長度 | 背景動畫 | 音效 |
| :---     | :---     | :---     | :--- |
| 和平     | 100%     | 辦桌、廟會、農耕 | 南管樂 |
| 不安     | 70%      | 人群聚集、竊竊私語 | 低沉鼓聲 |
| 緊張     | 40%      | 火光、塗鴉、爭吵 | 急促鑼鼓 |
| 爆發     | 0%       | 分類械鬥動畫、建築損毀 | 戰鬥音效 |

## 五、地理與環境系統

### 5.1 黑水溝（海象系統）

| 參數       | 說明 |
| :---       | :--- |
| **湍流值** | 0~100，影響航行安全 |
| **影響**   | 清軍援軍抵達天數、船隻損毀率、貿易利潤 |
| **季節**   | 颱風季（5-10月）湍流值基礎 +40 |
| **極端事件** | 湍流值 >80 時，航道完全封鎖 |

### 5.2 瘴氣系統（瘴雨蠻煙）

| 瘴氣等級 | 區域         | 效果 |
| :---     | :---         | :--- |
| 無       | 西部平原     | 無影響 |
| 輕度     | 北部丘陵     | 非在地勢力兵力 -5%/天 |
| 中度     | 中央山脈邊緣 | 非在地勢力兵力 -10%/天，行軍速度 -30% |
| 重度     | 深山獵場     | 非在地勢力兵力 -20%/天，行軍速度 -50% |

**對策**：
- 苦內那（金雞納樹）技術：研發後瘴氣傷害減半
- 漢方藥：短期免疫，持續 3 天
- 原民嚮導：僱用原民單位可完全無視瘴氣

### 5.3 動態災害系統

#### 5.3.1 地震（地牛翻身）
- **觸發**：分為「無害地震」與「有害地震」。無害地震每年約發生 500 次（現實中平均每 50 秒一次），僅造成畫面輕微搖晃；有害地震每年約發生 4 次。
- **效果（僅有害地震）**：
  - 依照震度（3.0 ~ 7.0）計算損傷。
  - 城牆防禦力減損（5% ~ 40%）。
  - 糧倉損毀（損失 5% ~ 40% 儲量）。
  - 民忠下降 5~20 點。
  - 高強度地震（>= 5.0）後可能出現新渡口或震開阻斷的山路（地形永久改變）。
- **流程**：有害地震發生前，全服導航列閃爍紅色預警。災後系統會標出受害庄頭，受害庄長可開啟【救災調度】小遊戲，庄民可點擊【捐獻】支援。

#### 5.3.2 颱風（季節風暴）
- **觸發**：限定在每年的 5-10 月（颱風季），每年隨機 2~4 次（或依機率平均 7 次）。
- **效果**：
  - 海上部隊強制撤退或受重創。
  - 隨機掃過特定路徑，沿途受害庄頭的農田產量歸零（糧食減半）。
  - 行軍速度降至 20%。
- **流程**：發生前 1 小時（遊戲時間）全服廣播預警。颱風登陸後，系統將發送受災名單，並為受災庄頭開啟救災小遊戲。

#### 5.3.3 瘟疫
- **觸發**：與瘴氣等級和人口密度相關
- **效果**：人口持續損失，需建造「藥鋪」對抗

### 5.4 時間流速引擎 (Time Engine)

為了讓萬人伺服器的歷史進程具有實感，遊戲內建獨立的時間流速引擎：

| 項目         | 設定值 |
| :---         | :--- |
| **起始基準年** | 公元 1600 年 1 月 1 日 凌晨 00:00 |
| **賽季長度**   | 真實世界 90 天 (一季) |
| **遊戲時長**   | 一個賽季推進遊戲內 300 年 (1600 ~ 1900) |
| **流速換算**   | 真實世界 1 秒 ≒ 遊戲內 1217.5 秒 (約 20 分鐘) |
| **四時辰制**   | 每日分為：早上 (06-11)、中午/下午 (12-17)、晚上 (18-23)、凌晨 (00-05) |

- **UI 同步**：伺服器每秒會發送 `TimeSync` 封包，客戶端導航列 (Navbar) 上會實時顯示當前架空年代與時段（例如：「1600年01月01日 早上」）。
- **機率計算**：所有天災的機率（地震、颱風）皆依據這個時間流速進行年化平均機率計算。

## 六、精選劇本設定

| :--- | :--- |
| **時代** | 大肚王國全盛期 |
| **主要勢力** | 大肚王國 vs 荷蘭東印度公司 vs 西班牙殘餘 |
| **可選角色** | 大肚王·甘仔轄·阿拉米 |
| **勝利條件** | 守護中部獵場，驅逐外來殖民勢力 |
| **特殊機制** | 火繩槍貿易——是否與敵人交易火器？ |

### 6.2 劇本二：1662 · 東寧覆亡與王國黃昏
| 項目 | 內容 |
| :--- | :--- |
| **時代** | 鄭成功退守台灣 |

| **特殊機制** | 「糧草燃燒計時器」——鄭軍存糧持續下降 |

### 6.3 劇本三：1721 · 鴨母王登基（核心劇本）
| 項目 | 內容 |
| :--- | :--- |
| **時代** | 朱一貴事件，全台大動亂 |
| **主要勢力** | 朱一貴（佔領府城）vs 清軍（退守澎湖）vs 大肚王國（趁機收復） |
| **勝利條件** | 義軍：援軍到前團結閩粵 / 清軍：藍廷珍成功登陸 / 大肚：收復失地 |
| **特殊機制** | 「內鬨計時器」——閩粵不合將導致義軍分裂 |

### 6.4 劇本四：1786 · 八卦旗下的反抗

### 8.1 資源類型

| 資源 | 取得方式 | 主要用途 |
| :--- | :---     | :---     |

| **鹿皮** | 獵場 | 海外貿易高價商品 |
| **樟腦** | 北部特產 | 高價出口 |
| **硫磺** | 北投、金山 | 火藥製造 |

### 8.2 海上貿易線
- 控制 80 海浬內的貿易點
- 可與「紅毛番（荷、英）」或「倭寇（日）」達成走私協議
- 走私成功：獲得滑膛槍等強力裝備
- 走私失敗：被清廷查獲，聲望暴跌

### 8.3 庄頭經濟
- 庄內商賈負責設定稅率（5%~30%）
- 稅率過高→民忠下降；稅率過低→建設資金不足
- 庄與庄之間可建立「牛車糧道」進行貿易

| :--- | :--- | :--- |

### 9.3 關鍵字情報感測

> *「聽聞南方有人在密謀大事...」*

- 非洩密式的趨勢提醒
- 增加策略感但不侵犯隱私
- 關鍵字清單由後端維護，可動態調整

## 十三、玩家體驗流程

### 13.1 新手引導

```
選擇陣營 → 分配至庄頭 → 教程：基礎操作
                           ↓
                    教程：內政經營
                           ↓
                    教程：第一次救災
                           ↓
                    教程：族群外交
                           ↓
                    自由遊玩
```

### 13.2 典型遊玩循環

```
登入 → 檢查庄頭狀況 → 分配精力值
  ↓
內政：開墾/建築/研發（消耗精力）
  ↓
外交：與其他庄頭/陣營互動
  ↓
軍事：部署防禦/發動攻擊
  ↓
事件：天災預警/族群緊張/情報接收
  ↓
精力耗盡 → 登出（或觀戰/社交）
```

### 13.3 賽季體驗範例：「朱一貴起義賽季」

1. **第一階段（平靜期）**
   - 各庄頭開墾，處理牛車調度，累積物資
   - 族群比例穩定，緊張氣氛儀火藥引信滿格

2. **第二階段（天災預警）**
   - 伺服器廣播「夏季強颱即將登陸」
   - 暫停軍事行動，進入救災小遊戲累積「災民忠誠」

3. **第三階段（轉折點）**
   - 地震震垮台南府城西側城牆
   - 朱一貴陣營收到情報，決定趁清軍救災時發動夜襲

4. **第四階段（族群爆發）**
   - 客家勇民因救災物資分配不均，緊張值爆表
   - 清軍後方爆發「分類械鬥」，三方大混戰
